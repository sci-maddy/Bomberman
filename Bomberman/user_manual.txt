# User manual
- This is clone of bomberman game for two players. 
- The aim of the game is to kill the other player with explosion of the bomb. 
- With explosions you can break crates in which there are sometimes bonuses. 
- Bonuses are: 
	- extra bomb - you can place more bombs at once
	- extra explosion - explosion of the placed bomb is greater
	- speed - you get temporary speed boost
- controls:
	- player1 - scientist
		w = up
		s = down
		a = left
		d = right
		space = place bomb

	-player2 - wizard
		- arrow up = up
		- arrow down = down
		- arrow left = left
		- arrow right = right
		- right ctrl = place bomb
